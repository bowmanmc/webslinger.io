Designing Beautiful Maps with D3.js

This is a talk I’ve given at the Dayton Data Visualization Meetup, Chicago Web Conf, and FOSS4G. It covers designing maps with D3.js, with each map building on the one before it. Starting out with a basic setup that includes retrieving geo-data from the server and drawing it on the map, the talk goes into using SVG and CSS tricks, and then utilizing D3 color scales and unique mapping projections to help you build the best map possible. The slides and source code I cover in the talk are available on github at https://www.github.com/bowmanmc/designing_maps.
